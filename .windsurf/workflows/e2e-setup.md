---
description: One-time setup for Playwright E2E testing (install deps and browsers)
---

## Steps

1. Install project dependencies:
   ```
   yarn install
   ```

2. Install Playwright browsers:
   ```
   npx playwright install
   ```

// Note: This workflow is for one-time local setup. Once done, you can run `yarn test:e2e` or use other e2e workflows.
